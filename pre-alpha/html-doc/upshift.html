Upshift

\upshift{value}

Use of this command is optional. Most documents will not need it. Cover artwork ignores it. If used, value is from 0 to 2. Decimal values allowed.

When you choose a template, Novelette automatically selects clearances, font size, and lines per page. The top of the header text will be about as far from the top edge of the page, as the bottom of them main body text is from the bottom of the page.

If you decide that the pages look like they should be shifted up a bit, then use this command. The value is multiple of line height.

The top of the printable area does not move. Instead, line-to-line spacing shrinks, thereby raising the bottom of the printable area.

Upshift does not change the line breaks, or the number of lines per page. So if you change the book trim size, you can also change upshift, as long as the new template keeps the same group number.

Note

Your P.O.D. Service may require space above the header, equal to space below the main body text. This is the default upshift 0. Or, you may be allowed to allocate space as you please, as long as clearance is enough.

Value 2 shifts the bottom of text up by 2 lines. That places the bottom of main text as far from the bottom of the page, as the top of main text (ignoring header) is from the top of the page.
