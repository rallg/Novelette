Media

The technical name for the PDF dimensions is media size, also called PDF page size. In most cases, the media size is the same as the book or markup trim size, so you do not need these commands.

There are two media settings. One of them applies only when the purpose is book. The other applies only when the purpose is markup. You may use one or the other setting, or both of them, or neither. They are independent. Both are ignored when the purpose is cover.

\media{markup}{width}{height}

Width and height must have units of mm, cm, or in. The dimensions cannot be less than the book trim size.

If you are printing a markup copy on a home or office printer, then you might find it useful to set the media size at the printer's paper size (usually USletter=8.5x11 inches, or A4=*****). Then the printed pages will appear centered on the paper, horizontally and vertically.

If you do not set the media size, then the printer may need further instructions from you. Or, it may guess.

Further discussion is on the markup page.

\media{book}{width}{height}

Normally, your P.O.D. Service will request that the media size be exactly the same as the book trim size. Then, do not use this command for the book.

But in a few cases, the P.O.D. Service will request that you center the trim size, horizontally and vertically, on a larger media size. Then you can use this command. The actual placement of your book trim is automatically written in Metadata, known as the TrimBox.

This does NOT print crop marks or other alignment marks.
