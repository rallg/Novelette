Purpose

\purpose{choice}

Every document must have a purpose. There are three choices:

  • book is the interior book block.
  • markup is the interior book block, for editing.
  • cover is for processing cover artwork.

Use book for the entire contents of the book. This prepares print-ready files in PDF/X format.

If markup, then the label MARKUP COPY is added to the top left corder of each page, along with lines 1in and 1cm long. Also, the PDF Metadata has MARKUP COPY added to the title. This is a convenient way to provide copies for editing, without confusing them with the finished product.

If cover, then the document can only be used for preparing the cover artwork as a PDF/X file. It is always a single page, with the entire cover and bleed.

In a typical workflow, use markup until you are satisfied with the entire book. Then, change markup to book and process the result.

Once you know the page count, your P.O.D. Service will provide the spine width. Then you are ready to work on the cover artwork.
